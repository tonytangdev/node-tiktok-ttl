# deno-tiktok-tts

## How to use

```bash
deno run --allow-net --allow-write index.ts en_us_001 "Hello world" "./my-file.mp3"
```

`en_us_001` being the voice to use.

`Hello world` being the text to be read.

`./my-file.mp3` being the file to save the audio to.
